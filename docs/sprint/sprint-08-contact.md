# Sprint 08 - Contact & Resume

## Objective

Create a professional Contact section that enables recruiters and hiring managers to connect through multiple channels while providing quick access to the latest resume.

- Designed Contact section with a strong call-to-action.
- Added availability status badge.
- Added responsive contact cards for:
  - Email
  - Phone
  - LinkedIn
  - GitHub
  - Location
  - Resume Download
- Implemented resume download functionality.
- Connected Hero "Contact Me" button to Contact section.
- Connected Hero "Download Resume" button.
- Added responsive layout for desktop and mobile.
- Added professional closing statement.
- Added application footer.
