# Project Vision

Build a modern, enterprise-grade personal portfolio that showcases my professional experience, technical expertise, projects, and services.

The portfolio should not look like a template. It should reflect the quality of software expected from a Senior Software Engineer with 15+ years of experience.

This project is also my learning journey for React + TypeScript + Tailwind CSS while applying enterprise software engineering principles.

# Development Philosophy

The project follows enterprise software engineering practices.

Principles:

- Clean Code
- Reusable Components
- Feature Based Architecture
- Strong TypeScript Typing
- Small Components
- One Responsibility Per Component
- Maintainable Folder Structure
- Scalable Design

Whenever possible:

Build once.
Reuse everywhere.
